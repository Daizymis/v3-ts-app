点击输入框，展示popup显示下拉数据

svalue:组件绑定字段即确认回调入参（item为对象）
slabel:下拉展示字段
options: 下拉列表
remote: 是否远程搜索
filterable: 是否过滤
placeholder
label

search(): 搜索事件回调
confirm(): 确认事件回调
change(): 选中数据改变回调
close(): 关闭popup回调
 <!-- field+select选择器使用示例 -->
      <field-select
        label="名称"
        placeholder="点击选择名称"
        remote
        s-label="ename"
        s-value="item"
        :options="employList"
        @change="changeValue"
        @confirm="value = $event"
        @search="getEmployeeList"
        @close="close()"
      ></field-select>
